🎨# AI-Powered Image Generator

An AI-powered image generation application that transforms simple user ideas into high-quality AI-generated images.

The application uses Groq LLMs for intelligent prompt enhancement and Google Gemini 2.5 Flash Image (via OpenRouter) for image generation, all wrapped in an interactive Streamlit web interface.

🚀# Project Overview

Creating high-quality AI-generated images often requires detailed and well-structured prompts. Most users provide short descriptions that may not generate the best results.

This project solves that problem by automatically enhancing user prompts using a Large Language Model (LLM) before sending them to an image generation model.

Example

User Prompt

A cat sitting on a mountain

Enhanced Prompt (Generated by Groq)

A majestic orange tabby cat sitting on a rocky mountain peak during sunrise, cinematic composition, golden hour lighting, ultra-detailed fur textures, wide-angle photography, dramatic clouds, realistic style, highly detailed environment, professional photography, 8K quality.

Generated Output

A visually appealing AI-generated image based on the enhanced prompt.

🔄 Workflow
Step 1: User Input

The user enters a simple idea.

Example:

A futuristic city floating above the clouds
Step 2: Prompt Enhancement

Groq LLM converts the simple idea into a detailed image-generation prompt.

Step 3: Image Generation

The enhanced prompt is sent to Gemini 2.5 Flash Image through OpenRouter.

Step 4: Image Display

The generated image is displayed in the Streamlit interface.

Step 5: Download Image

The user can download the generated image.
